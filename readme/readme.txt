Plugin for CudaText
Lite-analog of Scratches from JetBrains (C)

Plugin gives few commands in the menu "Plugins / Scratches":
"New" - Create a new snippet (scratch), with the lexer which you choose in the menu-dialog.
"List" - Show menu-dialog with available snippets (scratches), which you already created before.
"Remove" - Delete some available snippet (scratch), which you created before.
